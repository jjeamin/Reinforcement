- Bellman Expectation Equation(벨만 기대 방정식)

- Bellman Optimality Equation(벨만 최적 방정식)

## RL

- Monte Carlo Predict
  + predict(value)
  + model free
  + sampling
  + on policy

- Temporal Difference Predict
  + predict(value)
  + model free
  + sampling
  + bootstrap
  + on policy

- SARSA
  + control(policy)
  + model free
  + sampling
  + bootstrap
  + on policy

- Q Learning
  + control(policy)
  + model free
  + off policy
